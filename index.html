What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system for the web. It lets you lay content out in rows and columns, and has many features that make building complex layouts straightforward.

Key Benefits:
  • Two-dimensional layout (rows and columns)
  • Precise control over positioning
  • Responsive design made easy
  • Less dependency on floats and positioning

Basic Grid Properties

CSS - Basic Grid Setup 
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 equal columns */
  grid-template-rows: 100px 200px;    /* 2 rows with fixed heights */
  gap: 20px;                          /* Space between grid items */
}

/* Alternative column definitions */
.grid-container {
  grid-template-columns: repeat(3, 1fr);     /* Repeat 3 times */
  grid-template-columns: 200px 1fr 100px;    /* Mixed units */
  grid-template-columns: minmax(100px, 1fr) 2fr; /* Flexible sizing */
}

Grid Item Placement

CSS - Grid Item Positioning 
.grid-item {
  grid-column: 1 / 3;        /* Span from column 1 to 3 */
  grid-row: 2 / 4;           /* Span from row 2 to 4 */
}

/* Shorthand */
.grid-item {
  grid-area: 2 / 1 / 4 / 3;  /* row-start / col-start / row-end / col-end */
}

/* Named lines */
.grid-container {
  grid-template-columns: [start] 1fr [middle] 1fr [end];
}

.grid-item {
  grid-column: start / middle;
}

What are Grid Template Areas?

Grid template areas provide a visual way to define grid layouts using named areas. This makes your CSS more readable and maintainable.

CSS - Grid Template Areas 
.layout {
  display: grid;
  grid-template-areas: 
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-template-rows: 80px 1fr 60px;
  gap: 10px;
  height: 100vh;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

üìç CSS Positioning

Position Property Values

CSS - Position Properties 
/* Static (default) - follows normal document flow */
.static {
  position: static;
}

/* Relative - positioned relative to its normal position */
.relative {
  position: relative;
  top: 20px;    /* Move 20px down from original position */
  left: 30px;   /* Move 30px right from original position */
}

/* Absolute - positioned relative to nearest positioned ancestor */
.absolute {
  position: absolute;
  top: 0;
  right: 0;
}

/* Fixed - positioned relative to viewport */
.fixed {
  position: fixed;
  bottom: 20px;
  right: 20px;
}

/* Sticky - switches between relative and fixed */
.sticky {
  position: sticky;
  top: 0;
}

Z-Index and Layering

The z-index property controls the stacking order of positioned elements. Higher values appear on top.

CSS - Z-Index Examples 
/* Z-index only works on positioned elements */
.layer1 {
  position: relative;
  z-index: 1;
  background: red;
}

.layer2 {
  position: relative;
  z-index: 2;    /* This will be on top of layer1 */
  background: blue;
}

.layer3 {
  position: relative;
  z-index: 3;    /* This will be on top of both */
  background: green;
}

/* Negative z-index pushes element behind others */
.behind {
  position: relative;
  z-index: -1;
}
